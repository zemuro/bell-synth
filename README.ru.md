# Анализатор обертонов колокольных сэмплов

Утилита командной строки для анализа WAV-записей колоколов. Она находит спектральные обертоны, выводит их частоту, относительную амплитуду, ближайшую ноту в 12-тоновой равномерной темперации и отклонение в центах. По желанию можно построить визуализацию в matplotlib: спектрограмму участка затухания и график усреднённого спектра с подписанными обертонами.

## Возможности

- Загружает моно- и стереозаписи в формате WAV (проверено на 24 бит / 48 кГц).
- Приводит стереозапись к первому каналу.
- Пропускает настраиваемую атакующую часть в начале файла.
- Усредняет магнитудный спектр на участке затухания.
- Находит спектральные пики с настраиваемой выраженностью, расстоянием и сглаживанием.
- Сопоставляет каждый пик с ближайшей нотой и вычисляет отклонение в центах.
- Рисует спектрограмму и график спектра, сохраняет их в PNG.
- Работает без графического интерфейса — удобно для серверов и CI.
- Поддерживает INI-конфигурацию; флаги командной строки имеют приоритет.
- Документация на английском и русском языках.

## Быстрый старт

```bash
python -m venv venv
venv\Scripts\pip install -r requirements.txt

# Базовый анализ
python analyze_bell.py samples/bell.wav

# Сохранить визуализацию в PNG
python analyze_bell.py samples/bell.wav --plot-save --no-show --peak-count 12
```

## Настройка

Параметры по умолчанию хранятся в файле `analyze_bell.ini`. Отредактируйте его напрямую или сохраните текущую эффективную конфигурацию через `--save-config`.

```bash
# Сохранить текущую конфигурацию для редактирования
python analyze_bell.py --save-config my_config.ini

# Использовать свой конфиг
python analyze_bell.py samples/bell.wav --config my_config.ini
```

Подробнее — в [`docs/config.ru.md`](docs/config.ru.md).

## Устранение неполадок

**Не находит пики**
- Уменьшите `--prominence` или `--distance`, чтобы обнаружить тихие или близко расположенные обертоны.
- Проверьте, что `--min-freq` и `--max-freq` охватывают нужный диапазон.

**Слишком большой пропуск атаки**
- Уменьшите `--attack-skip-ms` или используйте более длинный сэмпл.

**PNG пустой или метки накладываются друг на друга**
- Ограничьте число подписей через `--n-labels`.
- Подберите `--spectrum-floor` или `--spec-floor`.

**Ошибка бэкенда matplotlib в headless-режиме**
- Всегда используйте `--no-show` вместе с `--plot-save`.
